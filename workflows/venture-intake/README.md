# Venture Intake Workflow

## Purpose

This workflow defines how a new venture is introduced into the system.

## Steps

1. capture the idea in second-brain
2. create a venture folder in second-brain/03_ventures
3. decide whether this is:
   - app
   - automation
   - content system
   - other
4. scaffold the project using app-factory if needed
5. create a venture wrapper in ai-factory/ventures
6. connect the venture to shared templates, prompts, and standards

## Output

A new venture is clearly defined across:

- second-brain
- repos
- ai-factory
