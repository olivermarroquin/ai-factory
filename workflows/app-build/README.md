# App Build Workflow

## Purpose

This workflow defines how I go from app idea to structured build.

## Steps

1. define the app in second-brain
2. use app-factory to scaffold the repo
3. use ai-agency-core templates and standards
4. create venture execution folder in ai-factory
5. document source references if reusing old code
6. build manually first
7. automate repeated steps later

## Rule

No agents should be introduced until the manual workflow is clear and repeatable.
